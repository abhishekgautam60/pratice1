such as- methodOverriding()

varible - First letter should be small and if it is contaning multiple words then first letter shoud be small and second letter should be capital.
such as - firstName

package - It should be lowercase letter as java, lang . If mutliple words are there then use (.) java.util, java.lang

CONSTANT - For using constant all letter should be in capital letter
